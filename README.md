# Financial News Summarizer

This project performs simple extractive and abstractive summarization on financial news articles.

### 🔍 Features
- Extractive summarization using TF-IDF + cosine similarity  
- Abstractive summarization using pretrained T5 transformer model  
- Sample financial article included  
- No training required  

### 📂 Project Files
- `summarizer.py` – main summarization script  
- `sample.txt` – sample financial news article  
- `requirements.txt` – dependencies  
- `README.md` – project documentation  

### 🚀 How It Works
1. The text is split into sentences  
2. TF-IDF identifies important sentences for extractive summary  
3. T5-small model generates abstractive summary  

### 📘 Usage
This is a static academic project for demonstration purposes.  
The code is included for reference and does not need to be executed.

### 📚 Dataset
You can place any financial article text in `sample.txt`.

### 🏁 Purpose
This mini-project is designed for academic submission and demonstrates basic NLP text summarization concepts used in finance.
